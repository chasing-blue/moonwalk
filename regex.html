Regex: search pattern for text

Single Character

  • \d : 0 - 9
  • \w : A - Z, 0 - 9 , a - z
  • \s : whitespace
  • . : any character

    • Position

  • ^ : beginning of the sentence
  • $ : end of the sentence
  • \b : word boundary

    • Quantifiers

  • * : 0 or more
  • + : 1 or more
  • ? : 0 or 1
  • {min, max} : a range from min to max
  • {number} : number of characters
