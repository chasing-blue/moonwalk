Regex: search pattern for text

home git 1 git 2 relax... Single Character

  • \d : 0 - 9
  • \w : A - Z, 0 - 9 , a - z
  • \s : whitespace
  • . : any character

    • Position
  • ^ : beginning of the sentence
  • $ : end of the sentence
  • \b : word boundary

    • Quantifiers
  • * : 0 or more
  • + : 1 or more
  • ? : 0 or 1
  • {min, max} : a range from min to max
  • {number} : number of characters
  • .*? : "?" makes the regex not greedy!!

    Character Classes
  • [abc] or [-.] : a/b/c or -/.
  • [a-z] : from a to z . Note: "-" is not literal character now
  • [^a-z] : NOT a to z. "^" is NOT here

    • Alternation
  • (net|org|com) : net/org/com

    • Capturing Groups
  • ([\w.]*)@\w+\.((com/org/edu)) : try to search email addresses. use $0, $1, $2 or \1 or \2 or \3 etc to refer to the group
